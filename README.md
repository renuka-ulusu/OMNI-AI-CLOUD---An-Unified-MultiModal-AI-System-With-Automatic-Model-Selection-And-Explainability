
# 🚀 OmniAI Cloud
Intelligent Unified Multi-Modal AI System with Automatic Model Selection and Explainability

📌 Overview

OmniAI Cloud is a unified Artificial Intelligence platform that can process images, text, and documents in a single system.

It automatically detects the input type and selects the appropriate AI models such as:

Computer Vision (CV)
Natural Language Processing (NLP)
Optical Character Recognition (OCR)

This eliminates the need for multiple tools and manual model selection.

🎯 Key Features
🔍 Automatic input type detection
🤖 Dynamic AI model selection
🖼️ Image analysis (classification, detection, captioning)
📄 Document processing (PDF, DOCX, OCR, summarization)
🌍 Multilingual translation support
😊 Sentiment analysis & Named Entity Recognition (NER)
📊 Explainable AI outputs with confidence scores
🌐 User-friendly web interface

💻 Installation
1️⃣ Clone the Repository
git clone [https://github.com/your-username/omni-ai-cloud.git](https://github.com/renuka-ulusu/OMNI-AI-CLOUD---An-Unified-MultiModal-AI-System-With-Automatic-Model-Selection-And-Explainability/blob/main/README.md)
cd omni-ai-cloud
2️⃣ Create Virtual Environment
python -m venv venv
source venv/bin/activate   # Linux/Mac
venv\Scripts\activate      # Windows
3️⃣ Install Dependencies
pip install -r requirements.txt
▶️ Run the Project
python app.py

Then open in browser:

http://127.0.0.1:5000/
